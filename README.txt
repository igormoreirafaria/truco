TRABALHO PRÁTICO DE REDES - SERVIDOR E CLIENTE
por Anna Luisa Aguilar, Igor Moreira, Igor Vilela

PARA COMPILAR O SERVIDOR DIGITE:
	
	$make

PARA COMPILAR O CLIENTE DIGITE:

	$make client

PARA EXECUTAR O SERVIDOR DIGITE:

	$./servidor

CASO O SERVIDOR ENTRE EM UM LOOP, E O LOOP DEMORE DEMAIS, BASTA DAR CTRL+C E EXECUTAR NOVAMENTE

PARA EXECUTAR OS CLIENTES(EM TERMINAIS DIFERENTES) DIGITE:

	$./cliente

APÓS EXECUTAR OS CLIENTES, É NECESSÁRIO INSERIR EM CADA TERMINAL OS
NÚMEROS 0, 1, 2, 3 EM CADA CLIENTE. OS CLIENTES 0 E 2 SÃO UMA DUPLA,
1 E 3 SÃO OUTRA.

AS CARTAS DEVEM SER DIGITADAS DA MANEIRA QUE SÃO EXIBIDAS NO TERMINAL(SEM AS ASPAS E SEM OS PIPES):

EXEMPLOS:	"|A c|", "|4 p|", "|K e|" => "|(LETRA OU NUMERO)ESPAÇO(CARACTER)|"
